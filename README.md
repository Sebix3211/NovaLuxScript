# NovaLuxScript
Język programowania, prosty i stworzony w pythonie
informacje: po wejściu w załączony plik pokaże wam sie GUI gdzie możecie wpisywać komendy. aktualna wersja to 1.2.0 i obsługuje komendy do pisania czegoś ( jak print w pythonie ) oraz możliwości liczenia działań. język jest w fazie rozwoju więc warto czekać na uktualizacje!
po wejściu do panelu pokaże wam sie kilka przykładów które warto skasować.
przy nowszych wersjach podam tutaj w README instrukcje różnych komend ale jak na razie pozostawiam wam możliwość odkrycia i nauki poprzez przykłady. 
język jest prosty i na poziomie coś pomiędzy scratchem a pythonem.. tak czy siak w okolicach pythona ale z powodu że NLS jest słąbo rozwinięty to komendy są łątwiejsze do nauczenia
