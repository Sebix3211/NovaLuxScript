# NovaLuxScript
Język programowania, prosty i stworzony w pythonie. NOWOŚCI: KOLOROWANIE SKŁADNI

po wejściu do panelu pokaże wam sie kilka przykładów które warto skasować.

intrukcja:
1. komenda/polecenie pisania tak smao jak w pythonie to po prostu print gdzie słowo które chcesz napisać też musi być w '' lub "" np. print "hello world"
2. żeby ustawić zmienną trzeba wykonać to samo co w pythonie ale na początku dać ,,set" np. set x = 1
3. jak narazie NLS przydaje sie jako prosty kalkulator ( no i wygląda jak z lat 90. ) dlatego by obliczyc coś należy użyć komendy calc np. calc 2 + 2, calc 2 * 2, calc 2 - 2, calc 2 / 2
4. aktualnie odbywają sie prace nad komendą if i else ale jak narazie nie działą za dobrze. pojawi się najwyżej w wersji 1.3.0 lub 1.4.0 
WAŻNE INFO: w przeciwieństwie do pythona tutaj jak dasz : to nie będzie czegoś takiego co ja nazywam ,,podkomendą" no wiecie o co chodzi.. w NLS jak robisz : to poprostu jakbyś kliknął w enter: normalnie przesunie cię na następną linijkę więc dla wygody by lepiej wyglądało wpisywanie komend jak robisz: to po nim przy następnym poleceniu warto dać 2-3 spację
